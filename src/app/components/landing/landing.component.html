Features

Speak with anyone, anytime

From productivity to entertainment, anyChatbot hosts all kind of personalities you can chat with. Programming pal? Check. That mercant NPC for your DnD campaign? Check. All bots ships with set personality and behaviour.

Build your own

Up for something different? Build a chatbot yourself in seconds. We ship a template (more soon!) for effective customization of ChatGPT behaviour, so expect less 'As a langeuage model' and prompt engineering fatigue and much more personaity and charisma.

Your API key, your terms

All you need to get started is an OpenAI API key and you are ready to go. Since is your API you can conveniently set how much token to spend on each reply and more (more setting coming soon!).

Free and Open Source

Although based on a paid service, you can fork the project on github and make it yours. However, it would be amazing if you step by github and help us develop it further. We have ana amazing feature roadmap just ahead!

Support us on GitHub!

It's the cheapest and most appreciated way to show your support for the project. We also welcome contributions to anyChatbotGPT. If you have any ideas or suggestions for new features, please let us know on GitHub.
